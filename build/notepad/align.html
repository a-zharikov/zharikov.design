Mixin Align

Миксин для абсолютного и фиксированного позиционирования элементов

Использование

@include top_center; - выровнит по верхнему левому краю

@include top_center(absolute,20px,40px); - выровнит по верхнему левому краю c отступоми

@include top_center(fixed,20px,40px); - выровнять по верхнему левому краю c отступом и фиксацией
